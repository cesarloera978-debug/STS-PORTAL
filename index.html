function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(["Form ID","Date","Customer Code","Customer Name",
      "Service Start","Service End","Part Number","Serial Numbers",
      "Lot Numbers","Mfg Date","Inspected","Approved","Rejected",
      "Reworked","Defect Summary","# People","Total Hours","Notes","Submitted By"]);
  }
  sheet.appendRow([data.formId,data.submitDate,data.customerCode,
    data.customerName,data.serviceStart,data.serviceEnd,data.partNumber,
    data.serialNumbers,data.lotNumbers,data.mfgDate,data.inspected,
    data.approved,data.rejected,data.reworked,data.defectSummary,
    data.numPeople,data.totalHours,data.notes,data.submittedBy]);
  return ContentService.createTextOutput(JSON.stringify({result:"ok"}))
    .setMimeType(ContentService.MimeType.JSON);
}

📋 Guía de Uso — Portal de Cliente
🔐
Paso 1 — Iniciar Sesión
En la pantalla de inicio, seleccione la pestaña 🏢 Client. Ingrese el código de cliente y la contraseña que STS Services le proporcionó. Ejemplo: código STS001
📊
Paso 2 — Ver Sus Reportes
Una vez dentro, verá el resumen general con el total de piezas inspeccionadas, aprobadas, rechazadas y horas trabajadas. Cada reporte muestra el número de parte, fecha y resultados completos.
⚠️
Paso 3 — Revisar Defectos
Cada reporte incluye la tabla de defectos con número de serie, lote/batch, descripción del defecto y cantidad. Si no hay defectos, aparecerá el mensaje ✅ Sin defectos registrados.
📸
Paso 4 — Ver Evidencia Fotográfica
Debajo de la tabla de defectos encontrará las fotos de evidencia tomadas durante la inspección. Estas muestran el estado físico de las piezas inspeccionadas.
📄
Paso 5 — Descargar PDF
Use el botón 📄 Descargar PDF en cada reporte para obtener una copia en PDF con todos los detalles, defectos y fotos. Guárdelo en su dispositivo o compártalo con su equipo.
❓
¿Reportes No Disponibles?
Si ve el mensaje 🔒 Reportes No Disponibles Aún, significa que su reporte está en revisión. Comuníquese con STS Services al portal.stslaredo.com para más información.
📞
¿Necesita ayuda? Comuníquese con STS Services directamente. Su código de cliente y contraseña son confidenciales — no los comparta con nadie que no sea parte de su equipo autorizado.
